[BahaaEldin0/NIH-Chest-Xray-14](https://huggingface.co/datasets/BahaaEldin0/NIH-Chest-Xray-14) -- a lot of different labels

[Pediatric Pneumonia Chest X-ray](https://www.kaggle.com/datasets/andrewmvd/pediatric-pneumonia-chest-xray) -- currently in use
